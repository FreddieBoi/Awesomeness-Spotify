Awesomeness-Spotifyby FreddieBoi

- How awesome are you?

Awesomeness is a very fancy Spotify App for determining how awesome you really are. For the real Home of the Awesomeness Spotify App, visit: http://awesomenessapp.herokuapp.com/

Dependencies

You will need the Spotify client.

Install

To install the Awesomeness Spotify App (on Windows 7):

  • Run the install.bat script.
  • Start Spotify and search for spotify:app:awesomeness .
